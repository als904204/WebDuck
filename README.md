# 웹툰 덕후들을 위한 사이트, WebDuck

---
## 소개

웹덕은 웹툰을 좋아하는 사람들의 웹툰 리뷰들을 모아놓은 커뮤니티 사이트입니다

웹툰 리뷰평을 보고 보고싶은 웹툰을 찾아봐요
